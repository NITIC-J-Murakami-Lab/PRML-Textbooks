「六面のサイコロを再現した関数を作ろう」#

import random

この関数は引数を受け取らず、返り値は面の数です。

例：

>>>dice()
3

import random

def dice():
    num = random.randint(1,6) 
    return num
dice()

「面の数を変えられるサイコロを実装した関数を作ろう」#

先程は六面に限定していましたが、今度は面の数を自由に変えられるようにしましょう。

目の数はn_rollで決定します。
それ以外は先程の関数と同様にしましょう。

def mutable_dice(n_roll:int)->int:
    num = random.randint(1, n_roll)
    return num

mutable_dice(10)

8
