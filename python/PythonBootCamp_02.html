「数字を逆順にしよう」#

与えられた数字を逆に並べ替える関数を作成して下さい。

例：4365->5634

以降、問題文の下に関数の雛形が与えられた場合はそれを改変して答えを作成してください。
また、作成した関数の実行結果まで示してください。

def reversed_int(x):
    """与えられた数字xを逆順に並び替えた数字resultを作成して返す
    """
    num = str(x)
    return num[::-1]

result = reversed_int(1234)
print(result)

「与えられた文書中の任意の文字列の出現回数を数える関数を作成しよう」#

与えられた文書に任意の文字列が何個出現するのか数える関数を作成して下さい。

例：一寸法師の歌詞中に「一寸法師」という文字列は何回出現するか

# ref http://www.worldfolksong.com/songbook/japan/issun-boshi.htm
sample_docs = """
指にたりない一寸法師
小さいからだに大きな望み
お椀の舟に箸のかい
京へはるばるのぼりゆく

打出の小槌京は三条の大臣どのに
抱えられたる一寸法師
法師法師とお気に入り
姫のお伴で清水へ

さても帰りの清水寺に
鬼が一匹現れ出でて
食ってかかればその口へ
法師たちまち踊りこむ

針の刀を逆手にもって
チクリチクリと腹つけば
鬼は法師を吐きだして
一生けんめい逃げていく

鬼が忘れた打ち出の小槌
打てばふしぎや一寸法師
ひと打ちごとに背がのびて
今は立派な大男
"""

def keyword_counter(keyword, sample_docs):
    """docs中に出てくるkeywordの出現回数を数える
    返り値のn_keywordは出現回数（int型）
    """
    n_keyword = sample_docs.count(keyword)
    return n_keyword# キーワードの出現回数

def count_keyword(key, docs):
  n = len(key)
  tmp = []
  counter = 0
  for c in docs:
    tmp.append(c)
    if len(tmp) == n:
      if key == "".join(tmp):
        counter += 1
      tmp = []
      continue
